# Project Constitution

## Technical Stack
- Backend: Laravel 11 (PHP 8.3+)
- Frontend: Vue 3 (Composition API, Vite)
- Database: PostgreSQL
- Styling: Tailwind CSS

## Architecture Principles
- API-First: Backend và Frontend giao tiếp thuần túy qua REST API.
- Single Responsibility: Mỗi Class/Component chỉ làm một việc duy nhất.
- Type Safety: Bắt buộc dùng Strict Types cho PHP và TypeScript (nếu có) cho Vue.

## Formatting & Standards
- Theo chuẩn PSR-12 cho Laravel.
- Vue components đặt trong `resources/js/Components`.
